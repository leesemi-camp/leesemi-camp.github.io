# 프로젝트 개요 (비-기술자용) (as of 2026-03-30)

이 저장소는 “우리 동네의 현안(문제/제보)을 지도에서 함께 보고, 선거사무원이 정리/관리할 수 있는” 정적 웹 페이지입니다.

## 누가 무엇을 할 수 있나요?

- **일반 방문자(공개)**
  - `/map/`에서 동(판교/운중/백현/대장 등) 경계와 “등록된 현안”을 지도/리스트로 확인합니다.
  - 현안을 클릭하면 위치와 간단한 내용(제목/분류/메모)을 확인할 수 있습니다.

- **선거사무원(편집 권한 staff)**
  - `/map/edit/`에서 Google 계정으로 로그인한 뒤, 권한이 확인되면 현안을 등록/수정/삭제할 수 있습니다.
  - 현안은 “좌표(지도에서 선택) + 제목/분류/내용” 중심으로 관리됩니다.

## 데이터는 어디에 저장되나요?

- 현안 데이터(Hotspot)는 Firebase Firestore에 저장됩니다.
- 동 경계/샘플 데이터(예: WFS XML, CSV)는 이 저장소의 `data/` 폴더에 정적 파일로 포함될 수 있습니다.

## 무엇이 ‘공개’이고 무엇이 ‘제한’되나요?

- 현안 데이터를 **읽는 것(열람)**은 공개일 수 있습니다(Firestore 규칙 설정에 따라).
- 현안 데이터를 **쓰는 것(등록/수정/삭제)**은 staff 권한이 있는 계정만 가능합니다.

## 앞으로의 확장(예: 경로/궤적)

현재는 “지점(점)” 중심의 현안(Hotspot)을 다룹니다.  
추가로 “경로/궤적(선)” 데이터를 지원하려면, 지도 편집 UI와 저장 구조를 별도로 확장해야 합니다(기술 문서 참고).
