# 성남시 공개 데이터 조사

이 문서는 대한민국 경기도 성남시와 관련된 통계정보 및 공개정보를 조사한 결과를 정리합니다. 우선순위는 대한민국 전자정부 누리집에서 제공하는 자료에 두고, 보완적으로 성남시청 공개정보 메뉴를 함께 확인했습니다.

## 조사 원칙

- **1순위**: 대한민국 공식 전자정부 누리집에서 제공하는 자료
- **2순위**: 성남시청 공식 공개정보 메뉴
- **정리 기준**: 데이터의 종류, 목적, 사용법, 현재 저장소와의 연결 가능성

## 우선 사용 포털

### 1. 공공데이터포털(data.go.kr)

이번 조사에서 가장 활용도가 높고 구조가 명확한 자료원입니다.

- 대한민국 공식 전자정부 누리집임이 명시되어 있습니다.
- 성남시 제공 파일데이터와 OpenAPI를 한곳에서 찾을 수 있습니다.
- 파일데이터는 로그인 없이 CSV 다운로드가 가능합니다.
- OpenAPI는 활용신청과 서비스키 발급 이후 REST 방식으로 사용할 수 있습니다.
- 일부 파일데이터는 포털에서 자동 변환된 OpenAPI 형태로도 활용할 수 있습니다.

### 2. 성남시청 공식 홈페이지

성남시청 메인 페이지 기준 다음 공개정보 메뉴가 확인됩니다.

- 사전 정보공표
- 재정정보
- 부서별공개자료실

이 중 `부서별공개자료실`은 실제 게시물 목록이 확인되었고, 주유소 품질검사 현황, 수질 결과 공개, 예산 집행 현황, 청년정책 시행계획 등 정기 공개 자료가 올라오고 있습니다. 반면 `사전 정보공표`, `재정정보`는 이 환경에서 직접 목록까지 추적되지 않아, 후속 수동 검토가 적절합니다.

## 데이터 유형별 정리

### 1. 인구·세대 통계

### 경기도 성남시_인구및세대_현황

- 출처: 공공데이터포털
- 형태: 파일데이터(CSV)
- 갱신 주기: 월간
- 제공 항목: 동별, 성별 인구, 총인구수, 19세 이상, 65세 이상, 세대수, 재외국민
- 목적:
  - 행정 수요 예측
  - 복지정책 수립
  - 도시계획 기초자료 확보
  - 동 단위 인구구조 변화 추적
- 사용법:
  - 가장 단순한 방법은 CSV 직접 다운로드입니다.
  - 시계열 비교가 필요하면 과거 월별 파일을 함께 수집해 누적 테이블로 재가공하는 방식이 적합합니다.
  - 지도 서비스에서는 동 경계와 결합해 choropleth 또는 label overlay로 활용할 수 있습니다.
- 현재 저장소와의 연결 가능성:
  - 이미 동 경계 데이터를 다루므로, 인구 데이터를 동 단위 통계 레이어로 붙이기 좋습니다.

### 2. 교통·보행 안전 위치 데이터

이 범주는 현재 저장소와 가장 직접적으로 연결됩니다. 모두 성남시 제공 OpenAPI이며, 위경도 기반 위치 데이터라는 점이 중요합니다.

- Base URL: `apis.data.go.kr/3780000/trfcCameraService`
- 엔드포인트: `GET /trfcCameraList`

- 목적:
  - 교통사고 다발 지역 분석
  - 단속 인프라 가시화
  - 교통 흐름 개선 정책 검토

- 사용법:
  - 공공데이터포털에서 활용신청 후 서비스키를 받아 REST 호출합니다.
  - JSON 응답을 선택하면 프론트엔드 지도 레이어에 바로 올리기 쉽습니다.
  - cluster 또는 density overlay에 적합합니다.
- 현재 저장소와의 연결 가능성:
  - 교통 오버레이 계층으로 바로 연결 가능합니다.

## 현재 저장소에서 우선 검토할 데이터

현재 저장소의 지도 구조와 현안 문맥을 고려하면 우선순위는 다음과 같습니다.

### 1순위: 지도에 바로 올릴 수 있는 위치 데이터

- 교통단속 카메라
- 교통약자 보호구역
- 보행자 전용도로
- 성남사랑상품권 가맹점현황

이들은 위경도 또는 위치 기반 응답이어서 기존 오버레이 구조와 결합하기 쉽습니다.

### 2순위: 동/구 단위 통계 레이어

- 인구및세대 현황
- 차량등록 차종별 통계현황
- 벤처기업현황

이들은 직접적인 위치점보다는 집계형 통계에 가깝습니다. 경계 데이터와 결합해 동/구 단위 비교 지표로 쓰는 것이 적절합니다.

### 3순위: 시민 안내 및 생활서비스 정보

- 분당구 행정복지센터 현황
- 분당구 정보화교육현황
- 성남시청 부서별공개자료실 문서들

이들은 지도의 주력 기능보다는 부가 정보 계층 또는 정책 보조 자료에 적합합니다.

## 사용 시 주의사항

- 공공데이터포털 OpenAPI는 대부분 활용신청과 서비스키 발급이 필요합니다.
- 파일데이터는 다운로드가 간단하지만, 자동화하려면 파일명 변경과 갱신 주기를 감안해야 합니다.
- 차량등록 통계는 법정동 기준이라 행정동 경계와 바로 맞지 않을 수 있습니다.
- 벤처기업현황은 주소가 인증 시점 기준일 수 있어 최신 주소 검증이 필요합니다.
- 성남시청 게시판형 공개자료는 정형 API가 아니므로 안정적 자동수집보다는 수동 검토 또는 별도 수집 로직이 적합합니다.

## 후속 조사 우선순위

1. 공공데이터포털에서 1순위 OpenAPI의 참고문서와 응답 스키마를 상세 확인
2. 인구 및 차량 통계를 현재 저장소 경계 체계와 어떻게 조인할지 검토
3. 성남시청 `사전 정보공표`, `재정정보` 메뉴를 브라우저 수동 탐색으로 추가 조사
4. KOSIS에서 성남시 기준 공식 통계표를 별도 확보해 공공데이터포털 자료와 상호검증
